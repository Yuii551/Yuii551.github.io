# Tan Yi Han - Gameplay Programming Portfolio

Source code and optimized media for my Unreal Engine gameplay programming portfolio.

The site showcases public work from:

- Brew & Bake
- AnkleBreaker

Internal company projects, proprietary systems, original game source code, and private development material are not included.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The website is deployed to GitHub Pages through the workflow in `.github/workflows/deploy.yml`.
